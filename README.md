MATLAB Data Science Portfolio: 7 Projects from Master's Coursework
This repository contains my complete Master's in Data Science coursework projects, all implemented in MATLAB. The collection demonstrates proficiency in statistical modeling, regression analysis, clustering techniques, and multivariate analysis through real-world datasets.

📁 Project Overview
#	Project	Focus Area	Key Techniques
1	Wafer Manufacturing Analysis	Multivariate Statistics	PCA, correlation matrices, covariance, ellipsoid volume, parallel coordinates
2	Soft Drink Brand Preferences	Categorical Data	Contingency tables, chi-square tests, Cramér's V, correspondence analysis, row profiles
3	CO₂ Emissions Prediction	Simple Linear Regression	Correlation, ANOVA, confidence intervals, F-statistics, residual analysis
4	Airbnb Pricing Analysis	Multiple Regression	Partial correlation, leverage plots, diagnostic residuals, model comparison
5	Restaurant Sales Forecasting	Advanced Regression	Dummy variables, stepwise selection, joint hypothesis testing, prediction intervals
6	Customer Segmentation	Hierarchical Clustering	Linkage methods (single, average, complete, Ward), dendrograms, cluster cutting
7	Coffee Shop Segmentation	K-Means Clustering	Manual algorithm implementation, Calinski-Harabasz criterion, adjusted Rand index, convergence tracking
